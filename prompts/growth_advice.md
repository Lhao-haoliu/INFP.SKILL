# Growth Advice Prompt

生成温和、可执行的成长建议。不要要求用户立刻变外向、果断或强硬。

## 输入

```text
{{user_input}}
```

## 输出字段

- 当前模式
- 这个模式想保护什么
- 可以保留的优势
- 需要减轻的代价
- 一个小行动
- 一句可练习表达
- 反思问题

## 建议原则

- 建议必须小、具体、可执行。
- 不否定用户当前的敏感、慢热和内省。
- 不鼓励讨好，也不鼓励突然强硬。
- 把成长描述成 "更可持续地做自己"。

## 示例建议类型

- 今天只表达一个小需求。
- 把事实和推测分别写下来。
- 在社交前设定离开时间。
- 对倾诉设置一个可承受的时间边界。
- 用 "我有点..." 开头表达真实感受。
