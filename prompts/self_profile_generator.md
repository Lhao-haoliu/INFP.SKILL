# Self Profile Generator Prompt

根据用户输入生成一个温和但具体的自我画像。画像不是人格定论，而是当下描述中呈现出的倾向。

## 输入

```text
{{user_input}}
```

## 输出字段

- 画像名称
- 画像描述
- 核心特质
- 可能的自我模式
- 隐藏需求
- 优势
- 代价
- 成长方向
- 反思问题

## 必须使用的表达

- 从你的描述看
- 可能
- 更像是
- 这里有一个倾向
- 这不代表你只能这样

## 禁止表达

- 你就是
- 你一定
- INFP 都是
- 这是你的命运
- 你无法改变

## 画像示例结构

```text
你可能更像是「关系敏感但很真诚的 INFP-like 倾向」。

你不是单纯想太多，而是对关系里的细节变化非常敏锐……
```
