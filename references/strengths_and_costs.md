# 优势与代价

| 特质 | 优势 | 代价 | 调整方向 |
| --- | --- | --- | --- |
| 共情力强 | 能敏锐感知他人情绪，更容易理解别人 | 容易把别人的情绪都背到自己身上 | 区分 "我理解他" 和 "我需要负责他" |
| 理想主义 | 有价值感、精神追求和审美标准 | 容易对现实失望，因不完美而迟迟不行动 | 把理想拆成小的现实行动 |
| 敏感细腻 | 能捕捉细节，也更懂关系中的微妙变化 | 容易从细节推导负面结论 | 把事实、推测、恐惧分开 |
| 想象力丰富 | 能创造意义、画面和可能性 | 容易沉浸在想象的关系或未来里 | 让想象服务一个具体行动 |
| 重视真实 | 不愿敷衍自己和他人 | 对浅层社交或虚假表达容忍度低 | 允许不同关系有不同深度 |
| 慢热 | 能谨慎选择关系，投入更稳定 | 容易被误解为冷淡，也可能错过开口时机 | 用一句话说明自己的节奏 |
| 善于倾听 | 让别人感到被理解和接住 | 容易成为长期情绪容器 | 设置倾听边界和恢复时间 |
| 重视意义感 | 能坚持有价值的方向 | 面对重复和现实妥协时容易无力 | 给日常任务连接一个小意义 |
| 内省能力强 | 能复盘自己，愿意成长 | 容易变成反复自责和内耗 | 给复盘设置出口：下一步做什么 |
| 不喜欢冲突 | 能减少关系伤害，保留温和 | 容易压抑边界和需求 | 练习温和但清楚地表达不舒服 |
| 情感投入深 | 关系真诚，记得细节 | 失望时恢复慢 | 给投入配套边界和自我照顾 |
| 审美和价值标准高 | 能创造有质感的作品或关系 | 容易被 "不够好" 卡住 | 允许草稿版先出现 |
