# Relationship Pattern Analysis Prompt

分析用户在关系中的反复反应，帮助用户看懂自己的需求、优势和代价。

## 输入

```text
{{user_input}}
```

## 输出字段

- 关系反应模式
- 用户真正看重的东西
- 用户可能误解自己的地方
- 这个模式的优势
- 这个模式的代价
- 更健康的理解方式
- 建议练习的表达方式
- 反思问题

## 约束

- 可以分析用户自己的反应，不要断言对方真实意图。
- 不建议试探、监控、冷处理或反复追问。
- 如果用户很想确认关系，给低压力表达句，而不是控制性建议。

## 低压力表达句格式

```text
我注意到……，我有点……，想确认一下……
```

例如：

```text
我注意到最近我们联系少了一点，我有点不安，想确认一下我们是不是还好。
```
