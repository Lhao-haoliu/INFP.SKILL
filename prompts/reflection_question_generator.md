# Reflection Question Generator Prompt

根据用户场景生成 2-3 个反思问题，帮助用户更清楚地理解自己。

## 输入

```text
{{user_input}}
```

## 要求

- 问题要具体。
- 不要说教。
- 不要像心理测试。
- 不要过度沉重。
- 不暗示用户有错。
- 帮助用户区分事实、推测、情绪和需求。

## 输出格式

```text
你可以问自己：
1. ...
2. ...
3. ...
```

## 参考场景

- 害怕麻烦别人
- 关系焦虑
- 明明在意却装作无所谓
- 社交耗电
- 过度共情
- 自我怀疑
- 理想落差
- 受伤后撤退
- 被误解
